# Contributing

Thank you for your interest in improving these notebooks.

## What We Accept

**Pull requests for:**
- Typos and spelling errors
- Broken links
- Code bugs that cause errors
- Incorrect outputs or results

**Issues for:**
- Reporting errors in code or text
- Broken notebooks or missing dependencies

## What We Don't Accept

- Pedagogical restructuring or reordering
- Alternative explanations or approaches
- New features or additional content
- Style or formatting preferences

The notebooks are designed to accompany a specific textbook and course. While we appreciate suggestions, major changes to content or structure are unlikely to be merged.

## Content Questions

For questions about the material itself, please use the course discussion forum rather than GitHub Issues.

## Submitting a Fix

1. Fork the repository
2. Create a branch for your fix
3. Make your changes
4. Submit a pull request with a clear description of the error and fix

Keep pull requests focused on a single issue.
